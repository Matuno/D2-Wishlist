# Monument of Triumph PvE Build Pack

Patch basis: Destiny 2 Monument of Triumph / Update 9.7.0, released June 9, 2026. Hotfix checks include 9.7.0.1 and 9.7.0.2. Bungie's support page lists Update 9.7.0.3 for July 7, 2026; no separate 9.7.0.3 patch-note article was discoverable during this pass, so the builds avoid interactions that were already flagged or fixed in the June hotfixes.

Scope: PvE endgame builds for raids, dungeons, Pantheon, Grandmaster-style activities, and hard solo play. PvP tuning is out of scope.

DIM link policy: every build file includes a generated DIM long-form import link. Verified public `dim.gg` links are retained beside the generated link when a current creator source exposes them. The generated link payloads are also written to `dim-loadouts.json` for auditing and regeneration.

## Reading Notes

- "Armor set" means the legendary Armor 3.0 series to farm around the exotic. The exotic itself should be treated as the build anchor.
- "Stats" use the post-Armor 3.0 names: Health, Melee, Grenade, Super, Class, and Weapons.
- Artifact choices are written as one active artifact plus exactly seven selected perks because DIM/imported loadouts cannot always be trusted to apply the relic state for every account.
- Long-form DIM links directly reference the subclass, selected subclass plugs, centerpiece exotic armor, named weapons from the build's Weapons section, and the seven artifact perk hashes. Account-specific armor and weapon copies still need review in DIM after import.
- Source confidence reflects currentness, DIM-link quality, and whether multiple independent sources point to the same core loop.
- Light.gg usage checks use Season 29 plus `Any PVE` and class filters. Lower-usage elements are kept for complete subclass coverage, but those files are labeled as coverage picks when the population data does not show them as top meta.
